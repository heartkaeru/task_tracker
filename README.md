# 📌 Alpha-Soft Task Tracker API

Backend-приложение для управления проектами и задачами на Django REST Framework.

Система реализует упрощённый таск-трекер (аналог Jira).

---

# 🚀 Описание проекта

Проект позволяет:

- создавать проекты
- добавлять участников
- создавать задачи внутри проектов
- назначать исполнителей
- отслеживать статус задач
- оставлять комментарии
- фильтровать задачи по параметрам

---

# ⚙️ Основной функционал

## 📁 Проекты
- создание проекта
- добавление участников
- доступ только участникам проекта

## 📌 Задачи
- создание задачи внутри проекта
- статус: todo / in progress / review / done
- приоритет: low / medium / high / critical
- назначение исполнителя
- дедлайн
- автор задачи

## 💬 Комментарии
- обсуждение задач
- привязка к конкретной задаче

## 🔍 Фильтрация
- по статусу
- по приоритету
- по исполнителю
- по проекту
- по дедлайну

---

# 🔐 Права доступа

- пользователь видит только свои проекты
- доступ к задачам только участников проекта
- автор задачи может редактировать описание
- исполнитель может менять статус и приоритет
- владелец проекта имеет полный доступ

# 📦 Установка и запуск

## 1. Клонировать проект

```
git clone <repository-url>
cd project
```

## 2. Создать виртуальное окружение
```
python -m venv venv
```

Активировать:

Windows:
```
venv\Scripts\activate
```
Mac/Linux:
```
source venv/bin/activate
```

## 3. Установить зависимости
```
pip install -r requirements.txt
```

## 4. Применить миграции
```
python manage.py makemigrations
python manage.py migrate
```

## 5. Создать суперпользователя
```
python manage.py createsuperuser
```

## 6. Запустить сервер
```
python manage.py runserver
```

# 📌 Особенности реализации
- кастомная система ролей через ProjectMembership
- object-level permissions (объектные права доступа)
- фильтрация задач как в Jira
- скрытые поля через HiddenField
- автоматическое назначение автора задачи
- бизнес-валидация через validate()
